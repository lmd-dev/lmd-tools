LMD-Processor is a small tool used to explain the principles of a tiny microprocessor. It combines an ALU, an accumulator, a RAM, a Program Counter and an external input.

Design Mode
When the design mode is enabled, you just have to set the processor states and then add this new instruction to the list.
When all the instructions of your program are defined, toggle to the Running Mode.

Running Mode
This mode process each instruction of the program.
